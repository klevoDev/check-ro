Как искать по классу (запись как выглядит в скобках)?

const buttonSubmitElement = document.querySelector('.form__button')

По классу обращение через точку

Как искать по id (запись как выглядит в скобках)?

const formElement = document.querySelector('#form-123e257')

По id обращение через #

Как искать по атрибуту (запись как выглядит в скобках)?

const inputAttrElement = document.querySelector('[type="text"]')

По атрибуту в квадратных скобках
